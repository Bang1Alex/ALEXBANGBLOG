# 批量删除node_modules

1. 创建一个普通文本(txt)
2. 把下面内容粘贴到☞创建的文本中(target_path修改记得修改)

```javascript
@echo off
chcp 65001 >nul
title 使用 rimraf 删除 node_modules

:: 设置目标路径
set "target_path=C:\Desktop\newWidget"

:: 进入目标目录
cd /d "%target_path%"

echo 正在使用 rimraf 删除所有 node_modules...
echo.

:: 使用 rimraf 删除
for /d /r . %%d in (node_modules) do (
    if exist "%%d" (
        echo 删除：%%d
        call rimraf "%%d"
    )
)

echo.
echo 删除完成！
pause
```

3. 把下面内容粘贴到☞创建的文本中
4. 修改后缀名为(.txt  → .bat)
5. 以管理员身份运行
